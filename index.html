Digital Image Processing - Lecture 1 (Full)

Lecture Content

A digital image is a representation of a two-dimensional image as a finite set of digital values called pixels.

Each pixel represents a gray level (intensity) or color.

Digitization implies approximation: a digital image is always an approximation of the real scene.

Common image formats:

  • 1 sample per point (B&W or Grayscale)
  • 3 samples per point (RGB Coloring System)
  • Improving pictorial information for human interpretation.
  • Processing image data for storage, transmission, or autonomous machine perception.
  • Low-Level: Image → Image (Noise removal, sharpening)
  • Mid-Level: Image → Attributes (Segmentation, object recognition)
  • High-Level: Attributes → Understanding (Scene understanding, autonomous navigation)

1920s: Bartlane cable picture transmission service.

1960s: Space race → processing images of the Moon (Ranger 7 probe).

1970s: Medical applications (CAT scans, Nobel Prize for tomography).

1980s-today: DIP explosion in:

  • Image enhancement/restoration
  • Artistic effects
  • Medical visualization
  • Industrial inspection
  • Law enforcement
  • Human-computer interfaces
